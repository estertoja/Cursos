Imagens em HTML

Para adicionar imagens usamos a tag:
- Nesta caso a imagens está na parta que o arquivo html:
≶ img src = "imagem.png" alt= "descricao da imagens" >

- Nesta caso a imagens está não parta que o arquivo html
≶ img src = "nome_pasta/imagem.png" alt= "descricao da imagens" >

- Também podemos carregar imagens externas, para isso devemos ter a url completa.
≶ img src = "nome_pasta/imagem.png" alt= "www.url.com.br" >

Favicon


O Favicon é o icone da Aba O código deve ser colocado no header: ≶ link rel = "shortcut icon" href= "nomefavicon.ico" type ="image/x-icon" >
