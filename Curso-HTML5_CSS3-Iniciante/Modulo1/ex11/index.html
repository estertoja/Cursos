    1. media = largura minima, no caso 1000px;
  • srcset = caso seja menos de 1000px irá carregar a imagem que for colocada como parametro
  • type = tipo da imagem
  • src = cao seja mais de 1000px irá carregar a umagem que for colocada como parametro
