  • Adicionando HiperLinks

    • Para criar Hiperlinks devemos criar ancoras:
    A tag ≶ a href=" url " > Texto ≶ /a ≶
    Google

  • Parametros

    • Podemos parametrizar comportamentos ao direcionar para um link

    1. _
      A tag ≶ a href=" url " target = "_blank" > Texto ≶ /a ≶
      Google
    2. _self = vai abrir o link em uma nova janela em branco

      3. A tag ≶ a href=" url " target = "_self" > Texto ≶ /a ≶
      Google
    3. _top = vai desfazer todos os frames e abrir o destino no navegador completo

      4. A tag ≶ a href=" url " target = "_top" > Texto ≶ /a ≶
      Google
    4. _parent = similiar ao caso _top em uma referencia á janela mãe

      5. A tag ≶ a href=" url " target = "_parent" > Texto ≶ /a ≶
      Google
    5. nome-do-frame = caso esteja usando frames, iniciar o nome da janela a abrir

      6. A tag ≶ a href=" url " target = "nome-do-frame" > Texto ≶ /a ≶
      Google
    6. hreflang = permite indicar qual é o idioma principal do site para onde o link está desviando o fluxo de navegação


  • Natureza do Destino


    • A tag ≶ a href=" url " target = "_self" rel ="next" > Texto ≶ /a ≶
    1. next = indica que o link é para a próxima parte do documento atual

      2. A tag ≶ a href=" url " target = "_self" rel ="next" > Texto ≶ /a ≶
      Google
    2. prev = indica que o link é para a parte anterior do documento atual

      3. A tag ≶ a href=" url " target = "_self" rel ="prev" > Texto ≶ /a ≶
      Google
    3. author = inidica que é um link para o site do autor do antigo atual

      4. A tag ≶ a href=" url " target = "_self" rel ="author" > Texto ≶ /a ≶
      Google
    4. external = inidica que é um link para outro site que não faz parte do site atual

      5. A tag ≶ a href=" url " target = "_self" rel ="external" > Texto ≶ /a ≶
      Google
